# Cluster

This project consists of a Next.js client and an Express server.

## Structure

- **/client**: Next.js application (Port 3000)
- **/server**: Express server (Port 4000)

## Getting Started

### Client

Navigate to the `client` directory and run:

```bash
cd client
npm run dev
```

### Server

Navigate to the `server` directory and run:

```bash
cd server
npm install
npm run dev
```
